# CtoAlgo :notebook_with_decorative_cover:

A simple bash script to convert C code to Algorithm-ish format


## Installation

```bash
sudo apt install clang-format
git clone https://github.com/Roshan-R/ctoaglo.git
cd ctoAlgo
chmod +x ./ctoalgo.sh
```
